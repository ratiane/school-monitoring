## рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ

рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃћрЃЉрЃў:

- **PDO Prepared Statements**: SQL Injection-рЃўрЃА рЃЌрЃљрЃЋрЃўрЃЊрЃљрЃю рЃљрЃфрЃўрЃџрЃћрЃЉрЃљ
- **password_hash()**: рЃърЃљрЃарЃЮрЃџрЃћрЃЉрЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ bcrypt-рЃўрЃЌ
- **CSRF Tokens**: Cross-Site Request Forgery рЃЌрЃљрЃЋрЃЊрЃљрЃАрЃ«рЃЏрЃћрЃЉрЃўрЃА рЃърЃарЃћрЃЋрЃћрЃюрЃфрЃўрЃљ
- **XSS Protection**: HTML escape рЃДрЃЋрЃћрЃџрЃљ output-рЃќрЃћ
